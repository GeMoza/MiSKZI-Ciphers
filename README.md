# MiSKZI-Ciphers

Учебный проект по дисциплине «Методы и средства криптографической защиты информации».

Проект реализует набор классических шифров, инфраструктуру для работы с вариантами из методических указаний и графический интерфейс на Streamlit для тестирования алгоритмов.

Текущая версия: **0.8.1**

Патч 0.8.1: исправлена поддержка локализованных символов (кириллица) в новых шифрах/настройках алфавита.

---

## Возможности проекта

### Реализованные шифры

- Атбаш  
- Шифр Цезаря  
- Скитала  
- Квадрат Полибия  
- Магический квадрат 4×4  
- Книжный шифр  
- Аффинный шифр  
- Двоичный код (1..33 → 8 бит)  
- Русская литорея  
- Шифр Виженера  
- Диск Альберти (параметризуемый)

Все шифры реализованы через единый контракт и автоматически подхватываются системой.

---

## Streamlit UI

### Playground (Песочница)

Позволяет:

- выбрать алгоритм
- просмотреть описание
- вводить ключ в режиме Form или Raw JSON
- зашифровать / расшифровать текст
- выполнить roundtrip (encrypt → decrypt)
- загрузить сохранённый вариант (read-only)
- загрузить free_text.txt (read-only)

Загрузка варианта или свободного текста не изменяет данные на диске. Изменение и сохранение выполняется только во вкладке Data Manager.

---

### Data Manager

Позволяет:

- просматривать и редактировать variants.json
- добавлять / удалять варианты
- валидировать структуру варианта
- запускать конкретный вариант
- редактировать free_text.txt

## Установка и запуск (Windows + conda)

### 1. Создать окружение

conda create -n miskzi-ciphers python=3.11 -y  
conda activate miskzi-ciphers  

---

### 2. Установить проект с UI

В корне репозитория:

python -m pip install -U pip  
python -m pip install -e ".[ui]"  

---

### 3. Запустить интерфейс

python -m streamlit run src/miskzi_ciphers/ui/app.py  

После запуска откройте в браузере:

http://localhost:8501
